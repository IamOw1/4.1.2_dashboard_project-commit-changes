# Быстрый старт COBA AI Drone Agent v4.1.2

## 🚀 Запуск системы

COBA AI Drone Agent — это полнофункциональная система автономного управления дронами с современным веб-дашбордом на React/TypeScript.

### 📋 Системные требования

- **Python**: 3.9 или выше
- **Node.js**: 18+ (рекомендуется 20+)
- **npm** или **bun** для управления зависимостями
- **RAM**: Минимум 4GB, рекомендуется 8GB+ для симуляторов
- **OS**: Linux, Windows 10+, macOS 12+
- **Браузер**: Chrome 90+, Firefox 88+, Safari 14+

### 📦 Установка

#### 1. Клонирование репозитория

```bash
git clone https://github.com/IamOw1/4.1.2_dashboard_project.git
cd 4.1.2_dashboard_project
```

#### 2. Установка Python зависимостей

```bash
# Рекомендуется использовать виртуальное окружение
python -m venv venv
source venv/bin/activate  # Linux/macOS
# или venv\Scripts\activate  # Windows

# Установка зависимостей
pip install -r requirements.txt
```

#### 3. Установка Node.js зависимостей

```bash
# Установка зависимостей фронтенда
npm install --legacy-peer-deps

# Альтернатива с bun (быстрее)
# bun install
```

### 🎯 Запуск системы

#### Вариант 1: Полная система (рекомендуется)

```bash
# Терминал 1: Запуск бэкенда (API + Agent)
python main.py api

# Терминал 2: Запуск фронтенда (дашборд)
npm run dev
```

**Результат:**
- **Дашборд**: http://localhost:8080
- **API**: http://localhost:8000
- **Документация API**: http://localhost:8000/docs

#### Вариант 2: Только фронтенд (демо-режим)

```bash
# Фронтенд с mock-данными
npm run dev
```

#### Вариант 3: Только бэкенд

```bash
# API сервер без фронтенда
python main.py api
```

### 🎮 Первый запуск: Что делать

1. **Открыть дашборд** в браузере: http://localhost:8080

2. **Проверить вкладки:**
   - **Телеметрия**: Должны отображаться графики (даже с mock-данными)
   - **Миссии**: Можно создать тестовую миссию
   - **Инструменты**: Включить/выключить модули
   - **Чат**: Спросить ИИ-помощника "Какие активные миссии?"

3. **Протестировать функции:**
   - Создать миссию на карте
   - Запустить симуляцию обучения
   - Проверить статус флота

### 🔧 Настройка для разработки

#### Переменные окружения

Создайте файл `.env` в корне проекта:

```bash
# API настройки
VITE_API_URL=http://localhost:8000

# Настройки карт
VITE_MAP_PROVIDER=osm  # osm, satellite, topography, dark

# Логирование
LOG_LEVEL=INFO

# Симуляторы (опционально)
AIRSIM_HOST=127.0.0.1
GAZEBO_WORLD=city
```

#### Конфигурация системы

Основные настройки в `config/config.yaml`:

```yaml
# Агент
agent:
  enabled: true
  decision_interval: 0.1

# Mesh сеть
mesh_network:
  enabled: true
  node_id: "drone_001"
  broadcast_port: 9000

# Инструменты
tools:
  - module: "object_detection"
    enabled: true
  - module: "slam_navigation"
    enabled: true
```

### 🚨 Устранение неполадок

#### Проблема: "Module not found" при запуске

```bash
# Очистка и переустановка зависимостей
rm -rf node_modules package-lock.json
npm install --legacy-peer-deps
```

#### Проблема: API недоступен

```bash
# Проверить, запущен ли процесс
ps aux | grep python

# Перезапустить API
python main.py api
```

#### Проблема: Фронтенд не компилируется

```bash
# Проверить версию Node.js
node --version

# Обновить npm
npm install -g npm@latest

# Повторная установка
npm install --legacy-peer-deps
```

#### Проблема: Карта не загружается

- Проверьте интернет-соединение
- Выберите другой провайдер карт в настройках дашборда
- Для оффлайн: загрузите тайлы заранее

#### Проблема: Обучение не запускается

- Проверьте установку PyTorch: `python -c "import torch; print(torch.__version__)"`
- Используйте mock-режим для демонстрации
- Установите симулятор (AirSim, Gazebo)

## Структура веб-дашборда

### 9 вкладок управления:

1. **Телеметрия** — реал-тайм мониторинг состояния дрона
2. **Миссии** — создание и управление полетными заданиями
3. **Команды** — быстрое управление дроном
4. **Инструменты** — статус 13 системных модулей
5. **Обучение** — обучение ИИ с подкреплением
6. **Флот** — управление группой дронов
7. **Камера** — видеопоток и обнаружение объектов
8. **События** — журнал событий и уведомления
9. **Бэкапы** — резервное копирование данных

### Плавающий чат с ИИ-помощником

Доступен на всех вкладках для быстрых консультаций и команд.
