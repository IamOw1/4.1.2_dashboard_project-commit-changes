# История изменений COBA AI Drone Agent

## v4.1.2 - Май 2026

### Интеграция dashboard-warm-heart

#### Добавлено

- 🔄 **Полная замена фронтенда** на dashboard-warm-heart
- 📊 **11 вкладок** вместо 9 (добавлены Инструкции, Настройки, Симуляция)
- 🗺️ **Интерактивные карты** с 4 провайдерами (OSM, Satellite, Topography, Dark)
- 🤖 **Улучшенный ИИ-чат** с smart-reply и контекстом миссий
- 🔧 **Расширенное API** с 50+ эндпоинтами
- 🎨 **UI на Radix UI** + Tailwind CSS v4
- ⚡ **TanStack Start** для маршрутизации
- 📱 **Полная адаптивность** desktop/tablet/mobile

#### Архитектурные изменения

- **Фронтенд**: React 19 + TypeScript + TanStack Router
- **Бэкенд**: FastAPI с mock-классами для демонстрации
- **Карты**: Leaflet с провайдерами и геозонами
- **Графики**: Recharts для телеметрии и обучения
- **Чат**: AIChat с floating UI и event-feed

#### Новые API эндпоинты

- `/api/v1/mesh/topology` - Топология mesh-сети
- `/api/v1/camera/*` - Управление камерой
- `/api/v1/detection/results` - Детекция объектов
- `/api/v1/events/*` - Журнал и фильтры событий
- `/api/v1/backup/*` - Резервное копирование
- `/api/v1/export/*` - Экспорт данных
- `/api/v1/memory/*` - Память агента
- `/api/v1/sub_agent/ask` - Запросы к суб-агенту

#### Документация

- 📖 **Полностью переписан README.md** с описанием всех вкладок
- 🚀 **Обновлен БЫСТРЫЙ_СТАРТ.md** с правильными командами
- 📚 **Создана documentation/current/** с подробными гайдами
- 🗂️ **Устаревшая документация** перемещена в documentation/old/

#### Технические улучшения

- Mock-классы для работы без зависимостей (PyTorch, OpenCV)
- HelpTooltip компонент для всех элементов управления
- Контекстные провайдеры (Mission, Ops, Safety)
- WebSocket поддержка для реал-тайм данных

### Структура вкладок

1. **Телеметрия** - Графики, карта, управление дроном
2. **Миссии** - Создание маршрутов, шаблоны, планирование
3. **Инструкции** - Правила, геозоны, протоколы безопасности
4. **Инструменты** - 13 модулей COBA с настройками
5. **Обучение** - DQN в симуляторах с графиками
6. **Флот** - Управление группой, mesh-топология
7. **Камера** - Видеострим, детекция, запись
8. **События** - Журнал с фильтрами, экспорт
9. **Бэкапы** - Создание/восстановление резервных копий
10. **Настройки** - Сенсоры, калибровки, тестирование
11. **Симуляция** - Управление виртуальными симуляторами
