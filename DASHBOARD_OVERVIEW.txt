🎯 ТИПИЧНЫЕ ДЕЙСТВИЯ:

  1. Просмотр статуса дрона:
     → Откройте HTML Dashboard
     → Нажмите "Connect"
     → Смотрите показатели в реальном времени

  2. Запуск сценария (например, патрулирование):
     → HTML Dashboard → нажмите карточку "Patrol"
     → Дрон автоматически начнет облет

  3. Навигация на точку:
     → Введите координаты X, Y, Z
     → Нажмите "Navigate"
     → Дрон полетит в эту точку

  4. Создание сложной миссии:
     → Откройте Streamlit на 8501
     → Вкладка "Missions"
     → Добавьте точки маршрута
     → "Launch Mission"

❓ ЧАСТЫЕ ВОПРОСЫ:

  Q: Дашборд не видит API
  A: Убедитесь что основной сервер запущен (python main.py)

  Q: Страница долго загружается
  A: Это нормально при первой загрузке (Streamlit инициализируется)

  Q: Хочу менять цвета
  A: Редактируйте :root переменные в web_interface/index.html

  Q: Как развернуть на сервере
  A: Смотрите RUN_DASHBOARD.md раздел "Production Deployment"
