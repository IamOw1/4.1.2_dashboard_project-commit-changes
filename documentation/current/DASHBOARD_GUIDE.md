# Руководство по вкладкам дашборда COBA AI Drone Agent v4.1.2

## Обзор

Дашборд состоит из 11 вкладок, каждая из которых предоставляет доступ к определенной функциональности системы. Все вкладки имеют унифицированный интерфейс с подробными подсказками (иконка "?") и адаптивным дизайном.

## 1. Телеметрия

Главная вкладка для мониторинга состояния дрона в реальном времени.

### Основные элементы

#### Графики в реальном времени
- **Высота**: Текущая и заданная высота, график за последние 2 минуты
- **Скорость**: Горизонтальная и вертикальная скорость с вектором движения
- **Батарея**: Уровень заряда с прогнозом времени полёта
- **Температура**: Температура компонентов дрона
- **Сигнал**: Уровень сигнала связи (Wi-Fi, mesh, радиоканал)

#### Панель состояния
- **Выбор дрона**: Дропдаун для переключения между дронами во флоте
- **Режим полёта**: MANUAL / AUTO / RTL / EMERGENCY
- **Статус моторов**: Температура и обороты по каждому мотору
- **Статус сенсоров**: GPS, камера, тепловизор, IMU и др.
- **Активные инструменты**: Список включённых модулей с их статусом

#### Интерактивная карта
- **Провайдеры**: Google Maps, Yandex Maps, OpenStreetMap, спутниковые снимки
- **Пользовательские карты**: Загрузка собственных тайлов
- **Отображение**:
  - Положение дронов с иконками и направлением
  - Маршруты миссий (пройденный/оставшийся путь)
  - Геозоны (безопасные/опасные зоны)
  - Обнаруженные объекты с всплывающими окнами

#### Кнопки управления
- **Пауза**: Приостановка текущей миссии
- **Вернуть домой (RTL)**: Автоматический возврат на точку взлёта
- **ARM/DISARM**: Активация/деактивация моторов
- **Старт/Стоп миссии**: Управление выполнением миссии

### Настройки автообновления
- Интервал: 1-10 секунд
- Включение/выключение обновлений

### Подсказки
Каждая кнопка имеет подробную подсказку с объяснением действия и возможных причин недоступности.

## 2. Миссии

Вкладка для создания, планирования и управления полётными заданиями.

### Создание миссий

#### На карте
1. Кликните на карте для добавления точки маршрута
2. Перетаскивайте точки для изменения маршрута
3. Правой кнопкой мыши для дополнительных опций

#### Шаблоны миссий
- **Патрулирование**: Автоматическое движение по периметру
- **Обследование**: Детальный осмотр заданной зоны
- **Доставка**: Транспортировка груза в точку назначения
- **Поиск**: Систематический поиск объектов в зоне

### Планирование маршрута

#### Учёт препятствий
- Автоматическое избегание зданий и деревьев
- Оптимизация высоты полёта
- Безопасные коридоры

#### Геозоны
- **Запрещённые зоны**: Красные полигоны (нельзя летать)
- **Ограниченные зоны**: Жёлтые полигоны (ограничения высоты/скорости)
- **Рекомендуемые зоны**: Зелёные полигоны

### Управление миссиями

#### Очереди миссий
- Последовательное выполнение нескольких миссий
- Приоритезация по важности

#### Запуск по расписанию
- Ежедневно в заданное время
- По событиям (например, при обнаружении объекта)

#### Условный запуск
- По датчикам (дым, движение)
- По командам от других дронов

### Импорт/Экспорт
- **Форматы**: JSON, KML, GPX, CSV
- **Валидация**: Проверка корректности маршрута перед импортом

## 3. Инструкции

Вкладка для управления правилами, законами и протоколами безопасности.

### Правила и законы

#### Добавление правил
- Текстовое описание ограничений
- Географические зоны применения
- Уровень приоритета (низкий/нормальный/высокий/критический)

#### Типы правил
- **Высотные ограничения**: Максимальная/минимальная высота
- **Скоростные лимиты**: Ограничения скорости в разных зонах
- **Временные окна**: Разрешённое время полётов
- **Погодные условия**: Ограничения по ветру, осадкам

### Геозоны

#### Создание геозон
1. Рисование полигона на карте
2. Выбор типа зоны (запрещённая/ограниченная/рекомендуемая)
3. Настройка параметров (высота, скорость, время)

#### Управление геозонами
- Включение/выключение зон
- Редактирование границ
- Импорт/экспорт зон

### Протоколы безопасности

#### Аварийные протоколы
- **RTL (Return To Launch)**: Возврат на точку взлёта
- **Land**: Экстренная посадка на месте
- **Hover**: Зависание на месте
- **Failsafe**: Автоматические действия при потере связи

#### Проверки перед запуском
- Валидация миссии на соответствие правилам
- Проверка состояния дрона
- Подтверждение оператора

## 4. Инструменты

Вкладка для управления 13 модулями COBA.

### Список инструментов

#### Категории
- **Навигация**: SLAM, EMEU, REPTOP
- **Сенсоры**: Object Detection, Thermal Vision
- **Управление**: Mission Planner, Autonomous Flight
- **Логистика**: Logistics, Precision Landing
- **Другое**: Geospatial Mapping, Object Detection

#### Для каждого инструмента
- Название и иконка
- Краткое описание
- Переключатель ВКЛ/ВЫКЛ
- Статус (готов/ошибка/загружается)

### Панель настроек

#### Основные настройки
- Включение/выключение
- Базовые параметры

#### Продвинутые настройки
- Детальные параметры конфигурации
- Калибровка сенсоров

#### Экспертные настройки
- Низкоуровневые параметры
- Отладочная информация

### Тестирование инструментов

#### Запуск теста
- Кнопка "Запустить тест"
- Безопасное тестирование без реального воздействия

#### Результаты тестирования
- **Готов к работе**: Инструмент функционирует корректно
- **Ошибка**: Описание проблемы с рекомендациями по исправлению

#### Примеры тестов
- **Object Detection**: Анализ тестового изображения
- **SLAM**: Симуляция движения в тестовом окружении
- **Navigation**: Проверка расчёта маршрута

## 5. Обучение

Вкладка для обучения ИИ с подкреплением (DQN).

### Выбор симулятора

#### Доступные симуляторы
- **AirSim**: Реалистичный симулятор от Microsoft
- **Gazebo**: ROS-совместимый симулятор
- **PyBullet**: Физический движок
- **SkyRover**: Кастомный симулятор COBA

#### Конфигурация окружения
- **Местность**: Город, лес, пустыня, горы
- **Погода**: Солнечно, дождь, снег, ветер
- **Время суток**: День, ночь, сумерки

### Задачи обучения

#### Доступные сценарии
- **Обход препятствий**: Навигация в сложной среде
- **Поиск людей**: Детекция и сопровождение
- **Скорость выполнения**: Оптимизация времени миссии
- **Экономия энергии**: Минимизация расхода батареи

### Управление обучением

#### Параметры обучения
- **Количество эпизодов**: 100-10000
- **Скорость обучения**: 0.001-0.1
- **Гамма (дисконтирование)**: 0.9-0.99
- **Размер пакета**: 32-512

#### Графики прогресса
- **Reward**: Награда за эпизод
- **Loss**: Функция потерь
- **Best Reward**: Лучший результат
- **Epsilon**: Исследовательский параметр

### Анализ результатов

#### Текстовый анализ
- Рекомендации по улучшению
- Анализ проблемных ситуаций
- Предложения по дообучению

#### Экспорт моделей
- Сохранение весов нейронной сети
- Форматы: PyTorch, ONNX, TensorFlow

## 6. Флот

Вкладка для управления группой дронов.

### Список дронов

#### Статусы дронов
- **Online**: Дрон подключен и готов
- **Offline**: Потеряна связь
- **Mission**: Выполняет задание
- **Charging**: Заряжается
- **Maintenance**: На обслуживании

#### Информация по дрону
- Уровень батареи
- Сила сигнала
- Текущая миссия
- Положение на карте

### Bulk-команды

#### Групповые команды
- **Старт миссии**: Для всех/выбранных дронов
- **Возврат домой**: RTL для группы
- **Зависание**: Hover на месте
- **Посадка**: Landing для всех

#### Выбор дронов
- Индивидуальный выбор
- Выбор по группам
- Выбор по статусу

### Формации

#### Типы формаций
- **Линия**: Прямая линия дронов
- **Круг**: Круговая формация
- **Клин**: Треугольная формация
- **Сетка**: Прямоугольная сетка

#### Параметры формации
- **Расстояние**: Между дронами (метры)
- **Высота**: Уровень полёта
- **Скорость**: Синхронная скорость

### Mesh-топология

#### Визуализация сети
- Узлы (дроны + оператор)
- Связи между узлами
- Качество соединений

#### Управление сетью
- **Шифрование**: Включение/выключение
- **Маршрутизация**: Автоматическая/ручная
- **Восстановление**: При потере узла

## 10. Настройки

Вкладка для конфигурации системы и сенсоров.

### Визуальные сенсоры

#### Камера
- **Разрешение**: 720p, 1080p, 4K
- **Частота кадров**: 30/60/120 FPS
- **Режимы**: День/ночь/тепловизор

#### Тепловизор
- **Диапазон температур**: -40°C до +500°C
- **Палитры**: Различные цветовые схемы
- **Чувствительность**: Настройка обнаружения

### Сенсоры и подключения

#### GPS/ГЛОНАСС
- **Точность**: Высокая/средняя/низкая
- **Количество спутников**: Минимум для фиксации

#### Связи
- **Wi-Fi**: Каналы, мощность сигнала
- **LoRa**: Дальность, скорость передачи
- **Mesh**: Параметры сети

### IMU и калибровки

#### Калибровка акселерометра
- **Автоматическая**: По команде
- **Ручная**: Шаг за шагом
- **Проверка**: Тест точности

#### Гироскоп и магнитометр
- Калибровка по тем же принципам

### Тестирование систем

#### Автоматическое тестирование
- Проверка всех сенсоров
- Диагностика связей
- Тест производительности

#### Результаты тестирования
- Статус каждого компонента
- Рекомендации по исправлению

Каждая важная кнопка имеет иконку "?" с подробной подсказкой:
- Что делает элемент
- Зачем он нужен
- Что происходит после нажатия
- Почему может быть недоступен

### Адаптивность
- **Desktop**: Полный интерфейс
- **Tablet**: Оптимизированный layout
- **Mobile**: Упрощённая версия с основными функциями

### Безопасность
- Подтверждение опасных действий
- Валидация вводимых данных
- Предупреждения о рисках

### Производительность
- Ленивая загрузка компонентов
- Кэширование данных
- Оптимизация рендеринга
