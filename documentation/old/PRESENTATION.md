# Презентация COBA AI Drone Agent (v4.0)

## Слайд 1. Идея
COBA AI Drone Agent — это “единая мозговая” архитектура управления дроном, где:
- Core Agent непрерывно принимает решения
- Sub-Agent помогает анализировать данные и формировать рекомендации
- Модули обеспечивают контроль моторов, сбор телеметрии и обмен сообщениями

## Слайд 3. Что делает система (пользовательский эффект)
- Даёт единый интерфейс управления через HTTP/WebSocket
- В режиме `all` поднимает API + непрерывный цикл агента
- Делает сбор и анализ данных полёта (OpenQ)

## Слайд 4. Команды управления (пример)
Пример через REST:
```bash
curl -X POST http://localhost:8000/api/v1/command ^
  -H "Content-Type: application/json" ^
  -d "{\"command\":\"TAKEOFF\",\"params\":{\"altitude\":10}}"
```

## Слайд 5. Телеметрия и WebSocket
- `GET /api/v1/telemetry` — телеметрия
- `ws://localhost:8000/ws/telemetry` — поток телеметрии

## Слайд 6. Что ещё предстоит (коротко)
- Полноценное выполнение миссий через API endpoint’ы (сейчас `mission/start` демо-возвращает идентификатор)
