## 🚀 Быстрый старт

### Способ 1: Автоматический (Рекомендуется)

**Linux/Mac:**
```bash
chmod +x start_dashboard.sh
./start_dashboard.sh
```

**Windows:**
```cmd
start_dashboard.bat
```

Это запустит все три сервиса одновременно:
- API на 8000
- HTML на 8080  
- Streamlit на 8501

### Способ 2: Ручной запуск

**Терминал 1 - API сервер:**
```bash
python main.py
```

**Терминал 2 - HTML интерфейс:**
```bash
cd web_interface
python -m http.server 8080
# Или: streamlit run --logger.level=error
```

**Терминал 3 - Streamlit приложение:**
```bash
streamlit run dashboard/app.py
```

### Способ 3: Docker Compose
```bash
docker-compose up
```

## 🔧 Конфигурация

### Изменение URL API

**В HTML интерфейсе:**
Просто вводите URL в поле "Connect to System"

**В Streamlit приложении:**
```bash
# Способ 1: Переменная окружения
export API_URL="http://your-api:8000"
streamlit run dashboard/app.py

# Способ 2: Отредактируйте в файле
# dashboard/app.py строка 50:
API_URL = "http://your-api:8000"
```

### Смена темы Streamlit

Отредактируйте `.streamlit/config.toml`:
```toml
[theme]
primaryColor = "#00d4ff"
backgroundColor = "#0f1419"
secondaryBackgroundColor = "#1a1f2e"
```

## 🐛 Решение проблем

### Порт занят
```bash
# Проверить процесс
lsof -i :8080

# Использовать другой порт
python -m http.server 8081
```

### API не доступен
```bash
# Убедитесь что API запущен
curl http://localhost:8000/health

# Если нет, запустите
python main.py
```

### Streamlit не запускается
```bash
# Очистить кэш
streamlit cache clear

# Переустановить
pip install --upgrade streamlit
```
