============================================================
 ТЕХНОЛОГИИ
============================================================

Frontend:  React 19 + TypeScript + TanStack Start + TanStack Router
UI:        Tailwind CSS v4 + Radix UI (45+ компонентов)
Карты:     Leaflet с 4 провайдерами (OSM, Satellite, Topography, Dark)
Графики:   Recharts для телеметрии и обучения
Backend:   Python 3.9+ + FastAPI + Uvicorn
API:       50+ эндпоинтов REST + WebSocket (опционально)
База:      SQLite для логов и бэкапов
